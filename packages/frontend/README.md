# NTG Login Frontend

React frontend application for NTG Login profile management.

## Installation

```bash
cd packages/frontend
npm install
```

## Development

```bash
npm run dev
```

App will run on http://localhost:5173

## Build

```bash
npm run build
```

## Environment Variables

Create `.env` file:

```
VITE_API_BASE_URL=http://localhost:3000
```

## Features

- ✅ Login/Logout
- ✅ Dashboard with statistics
- ✅ Profile management (CRUD)
- ✅ Proxy management (CRUD)
- ✅ Session management
- ✅ Job monitoring
